# Testplan – ShopSphere Version 1.0

## Testumgebung
Windows 11, Chrome oder Edge, Java 21 und Maven 3.9+.

## Umfang
35 automatisierte Tests: 22 UI-, 9 API- und 4 Datenbanktests.

## Erwartetes Ergebnis
30 Tests bestehen. Fünf Produktfehler in Filter- und Sortierfunktionen werden
reproduziert.

## Freigabeempfehlung
Keine Produktionsfreigabe, bevor BUG-001 bis BUG-005 behoben und erneut getestet
wurden. Für Portfolio- und Demonstrationszwecke ist die Version geeignet.
